For HW3, we mainly use the following packages:
  OpenCV(*opencv-python)
  Numpy
  matplotlib
  tqdm

Although they should be installed already in TA's environment, in case here is the command on how to install the packages:
Inside your conda virtual environment, go to HW3 folder and run command "conda install --yes --file requirements.txt".


*How to run the code:
Inside your virtual environment, go to HW3_111065524 folder, cd to folder "1" to run "python HW3-1.py" and cd to folder "2" "python HW3-2.py" respectively.
Press enter to close the current image window and continue the code.
