For HW2, we mainly use the following packages:
  OpenCV(*opencv-python)
  Numpy
  matplotlib

Although they should be installed already in TA's environment, in case here is the command on how to install the packages:
Inside your conda virtual environment, go to HW2 folder and run command "conda install --yes --file requirements.txt".


How to run the code:
Inside your conda virtual environment, go to HW2_111065524 folder, and run "python hw2_1.py" and "python hw2_2.py" respectively.
Press enter to close current image window and continue the code.
